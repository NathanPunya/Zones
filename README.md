# Zones

**What if Strava and Monopoly had a baby, but the baby was mostly a map?**

Zones is an iOS app for people who look at a neighborhood and think *“that could be mine.”* Run around, draw loops on a Google Map, and **claim patches of turf** like you’re marking a tree (but legally fuzzier). There’s a **leaderboard** so you can flex on friends, **streaks** so guilt works as motivation, and an **AI route generator** that asks Google for walking loops so you don’t have to plan—just slide a “how spicy is this loop” level and try not to anger the Directions API.

Health stuff hooks in, notifications nag you about streaks, and if you skip the Firebase plist you’re basically playing **offline fantasy cartographer**. Throw in a `.env` with a Maps key or the app gets passive-aggressive in tiny gray text.

That’s the gist. Go touch grass—then come back and claim it.
